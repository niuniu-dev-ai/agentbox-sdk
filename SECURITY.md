# Security Policy

Please do not open public GitHub issues for vulnerabilities.

For SDK security issues, use GitHub private vulnerability reporting if it is
available on this repository. If private reporting is unavailable, contact the
maintainers through the contact path listed on
[agentbox.niuniu.dev](https://agentbox.niuniu.dev/).

Useful reports include:

- affected SDK package and version;
- a minimal reproduction;
- expected and actual behavior;
- any relevant request or response metadata with secrets removed.

Never include live identity tokens, grant tokens, admin tokens, or other
secrets in a report.
